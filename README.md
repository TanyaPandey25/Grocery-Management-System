👩🏻‍🔧How to Run the Program: 
 
1. 📩Download/Clone the Project: Ensure one has the Python script and 
required files (grocery_list.bin, stores_list.bin, customer_list.bin, 
feedback_list.bin, password.txt). 
2. 🔮Install Python: Make sure Python 3.x is installed on the machine. One can 
download Python from python.org . 
3. 🪄Run the Program: 
-> Open the terminal or command prompt. 
-> Navigate to the folder where the script is saved. 
-> Run the script using the command: 
      python grocery_store_management.py 
 
-> The program will ask for user input. Select either Admin or Customer to proceed. 
 
👩🏻‍💻Admin Interface: 
 
The Admin can perform various tasks such as: 
-> Add Grocery Items: Add new items to the inventory with details like name, category, price, 
quantity, manufacturing date, and expiry date. 
-> Display Grocery Items: View a list of all available items in the store. 
-> Check Expired Items: View a list of items that have expired based on the entered date. 
-> New Arrivals: View newly added items within the last 30 days. 
-> Add Store: Add a new store with a name and location. 
-> View Sales Data & Profit/Loss: View sales data and profit/loss reports for each store. 
-> Change Admin Password: Modify the password for the admin interface. 
-> View Feedback: View feedback left by customers on different grocery items. 
 
 
👩🏻‍🦰Customer Interface: 
 
The Customer can: 
-> View Available Items: Browse the available grocery items in the store. 
-> Buy Groceries: Purchase items by selecting them and entering the required quantity. 
-> View Customers: Check the list of all registered customers. 
-> Leave Feedback: Leave ratings and comments on purchased items.
